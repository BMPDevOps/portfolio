안녕하세요! 멀티 IDC 환경에서 안정적이고 확장 가능한 인프라를 구축하며 지속 가능한 시스템을 만들어가는 인프라 엔지니어 박보민입니다.

제이메타 ITO팀에서 1년 3개월간 실무 경험을 쌓았으며, Windows Server 및 Linux 시스템 운영/유지보수, PowerShell 자동화 스크립트 개발, DLP/NAC/ECM 보안 솔루션 운영 등의 업무를 담당했습니다.

개인적으로는 2019년부터 5년간 Homelab을 운영하며 ESXi 2대 + Proxmox 1대 구성과 Ceph 분산 스토리지를 관리하고 있습니다. 이를 통해 엔터프라이즈급 가상화, 컨테이너 오케스트레이션, 스토리지 관리 등 현대적인 인프라 기술을 실습하고 있습니다.

핵심 인프라 역량

  • 컨테이너 오케스트레이션: Kubernetes, OpenShift 클러스터 구축 및 관리
  • 서버 가상화: VMware vSphere, Proxmox VE 기반 고가용성 환경 구성
  • 분산 스토리지: Ceph 클러스터 구축 및 RBD/CephFS 운영
  • 인프라 자동화: Ansible, Terraform을 통한 Infrastructure as Code
  • CI/CD 파이프라인: GitOps 기반 배포 자동화 환경 구축

🏠 HOMELAB 프로젝트

2019.06 - 현재 (5년간 운영) 플래그십 프로젝트

🎯 프로젝트 목표

개인 학습 및 다양한 기술 스택 테스트를 위한 엔터프라이즈급 인프라 환경 구축

🏗️ 인프라 아키텍처

컴퓨팅: ESXi 서버 2대 + Proxmox VE 서버 1대
스토리지: Ceph 분산 스토리지 클러스터
네트워킹: 10GbE 네트워크 및 2.5GbE 네트워크 구축
가상화: vSphere 클러스터 + KVM 하이브리드 환경

⚡ 내가 담당한 주요 업무

  • ESXi 서버 2대를 VMware vSphere로 클러스터링하여 고가용성 환경 구성 - vMotion, HA, DRS 설정으로 무중단 서비스 보장
  • Ceph 기반 분산 스토리지 클러스터 구축으로 확장 가능한 데이터 관리 환경 확보 - OSD, MON, MGR 노드 구성 및 RBD로 Kubernetes StorageClass로 연결
  • OKD 클러스터 구축 및 ArgoCD GitOps 배포 자동화 환경 구성 - 마스터 3노드 + 워커 3노드 클러스터 운영
  • Ansible/Terraform을 이용한 Infrastructure as Code 구현 - 인프라 프로비저닝 및 설정 관리 자동화
  • Prometheus + Grafana 모니터링 스택 구축 - 시스템 메트릭 수집 및 대시보드 구성
  • Harbor를 활용한 레지스트리 관리 - 컨테이너 이미지 관리 및 Helm 차트 관리

📈 기술적 성과

  • 99.9% 가동 시간 달성 (연간 8시간 이하 다운타임)
  • 자동화를 통한 운영 작업 80% 시간 단축
  • 스토리지 풀 안정적 운영 (데이터 손실 0건)
